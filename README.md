# hackathon-n8n

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.js
```

auto run nodemon 
```bash
bun run start
```

after create new slash commands 
```bash
bun run deploy-commands
```

This project was created using `bun init` in bun v1.1.43. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
